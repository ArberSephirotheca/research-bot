  - **Tensor Cores (GPU) vs systolic arrays (TPU):** Tensor Cores deliver high peak MAC throughput via many small, independent MMA units (warp-level tiles) that rely on high-bandwidth memory + shared-memory tiling; TPUs use a large 2D systolic array where data is streamed/reused spatially, making utilization more sensitive to matrix shape and dataflow alignment.
  - **Matmul behavior & utilization:** GPUs typically hit ceilings when matmul tiles map cleanly to Tensor Core fragment sizes and when shared-memory/L2 reuse keeps the MMA pipelines fed; otherwise performance becomes memory/latency bound. TPUs tend to sustain high utilization on large, regular GEMMs but can underutilize on “skinny”/irregular dimensions due to array fill/drain and padding.
  - **Practical GPU takeaway:** To approach Tensor Core
